## 🧩 Motivering av tekniska val

- ⚛️ **React + TypeScript** — ger typsäkerhet, bättre autokomplettering och färre buggar.
- ⚡ **Vite** — extremt snabb utvecklingsmiljö med hot module replacement.
- 🎨 **Tailwind CSS** — utility-first approach med konsekvent designsystem via CSS-variabler.
- 🎭 **Framer Motion** — professionella animationer med deklarativ syntax.
- 🌐 **Custom i18n** — lättviktig flerspråkslösning utan tunga dependencies.
- 🌙 **Theme System** — CSS-variabler som automatiskt anpassar alla komponenter.

## 📂 Projektstruktur

```
/src
  /assets        → bilder och grafik
  /components    → React-komponenter
    /sections    → Sektioner (Hero, About, Skills, etc.)
    /ui          → Återanvändbara UI-komponenter
  /hooks         → Custom React hooks
  /lib           → Utilities, i18n, tema
  /pages         → Sidkomponenter
  App.tsx        → Huvudkomponenten
  main.tsx       → Entry point
  index.css      → Design system (CSS-variabler)

tailwind.config.ts → Tailwind-konfiguration
index.html         → Vite HTML-template
package.json       → Dependencies & scripts
```

## 🧪 Installation & körning (lokalt)

### 1. Klona repot
```bash
git clone <repo-url>
```

### 2. Installera dependencies
```bash
npm install
```

### 3. Starta utvecklingsservern
```bash
npm run dev
```

### 4. Öppna projektet
Vite visar en lokal URL, t.ex.: `http://localhost:8080/`
